# Blitzblank — Releases

Öffentliche Releases von **Blitzblank**, der deutschsprachigen
Windows-11-Wartungs-App für die ganze Familie: Ein-Klick-Aufräumen mit
Vorschau, ehrliche Stabilitäts-Diagnose und MCP-Schnittstelle für
KI-Agenten — ohne Scareware, ohne Cloud, ohne Konto.

## Download

**[⬇ Blitzblank für Windows 11 laden](https://github.com/blackdiamond6390/blitzblank-releases/releases/latest/download/Blitzblank-Setup.exe)**

Der Link lädt immer den Installer der neuesten Version
(`Blitzblank-Setup.exe`). Alle Versionen samt Signaturen stehen unter
[Releases](https://github.com/blackdiamond6390/blitzblank-releases/releases).

## Hinweise

- **Windows SmartScreen:** Die App ist (noch) nicht code-signiert. Beim
  ersten Start ggf. „Weitere Informationen" → „Trotzdem ausführen" wählen.
- **Auto-Update:** Installierte Apps aktualisieren sich über die hier
  veröffentlichte `latest.json` (Tauri-Updater, minisign-signiert).
- Jedes Release enthält: `Blitzblank_<version>_x64-setup.exe`, dessen
  `.sig`-Signatur, `latest.json` und das versionslose Alias
  `Blitzblank-Setup.exe`.
